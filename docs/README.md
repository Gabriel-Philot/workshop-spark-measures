# Documentation index

Short index for repository-level documentation.

## Folders

- `architecture/`: platform/runtime patterns that explain how labs are executed.
- `design/`: current design decisions used by the workshop implementation.
- `research/`: source-backed investigations that support design decisions.
- `archive/`: older research, verdicts, and planning notes kept for historical context.

## Current entry points

- `architecture/runtime-pattern.md`: lab runtime lifecycle and measured-workload boundary.
- `design/data-generator-design.md`: current Spark-native generator tradeoff.
- `design/spark-worker-topology.md`: local Spark worker topology and sizing.
- `research/data-generator-research.md`: evaluated generator engines, relational
  data strategies, scale assumptions, and source references.
