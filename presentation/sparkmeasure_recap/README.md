# sparkMeasure recap presentation

Static HTML/CSS/JavaScript presentation that connects Spark fundamentals,
observability, policies, telemetry contracts, and responsible automation.

## Run locally

From the repository root:

```bash
python3 -m http.server 28504 --bind 0.0.0.0 --directory presentation/sparkmeasure_recap
```

Open [http://127.0.0.1:28504/](http://127.0.0.1:28504/).

The main entry point is `index.html`. Use the arrow keys to navigate and `N` to
toggle speaker notes.
