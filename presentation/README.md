# Presentation materials

Static classroom presentation assets for the sparkMeasure workshop.

## Available presentations

- `sparkmeasure_origin/`: origin story of sparkMeasure, covering CERN, data scale, Spark bottlenecks, and why performance observability became necessary.
- `sparkmeasure_mecanix/`: technical mechanics of sparkMeasure, covering Spark runtime traces, listener events, StageMetrics, TaskMetrics, and observability tradeoffs.

Each folder is self-contained and can be served with a simple static HTTP server.
